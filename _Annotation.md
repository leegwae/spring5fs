# Annotation



### org.springframework.context.annotation

| 애노테이션      | 설명                                                         | 참고 |
| --------------- | ------------------------------------------------------------ | ---- |
| `Bean`          | 빈 설정 메서드에 붙이는 annotation. 해당 메서드가 생성한 객체를 Bean 객체로 등록한다. | 2    |
| `Configuration` | 해당 클래스가 스프링 컨테이너(설정 정보를 담고 있음)을 표기한다. | 2    |
| `Autowried`     | 스프링 설정 필드에 해당 타입의 빈 객체를 자동으로 할당한다.  | 3.8  |
